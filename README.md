# TFILM New Site

Clean Astro architecture for a lightweight cinematic portfolio website.

## Commands

```bash
npm install
npm run dev
npm run build
npm run preview
```

## Structure

- `src/layouts`: shared document/page layouts.
- `src/components`: reusable UI and media components.
- `src/data`: project metadata and navigation data.
- `src/pages`: static routes and project route scaffolding.
- `src/styles`: global design tokens and base styles.
- `public/assets`: future web-ready assets copied from curated source assets.

The Squarespace backup is intentionally not imported.
